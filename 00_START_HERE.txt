CRITICAL FINDINGS (TL;DR)
================================================================================

8 EXPLOITABLE VULNERABILITIES (No RSA Break Required):

1. [CRITICAL] SMC Instance Not Validated
   File: acr_dma_ga100.c
   Issue: falconInstance used in register address without bounds check
   Impact: Arbitrary MMIO write, DMA misdirection

2. [CRITICAL] Dependency Map Array Bounds
   File: acr_wpr_ctrl_ga10x.c
   Issue: depfalconId used as array index without validation
   Impact: Out-of-bounds read, information disclosure

3. [CRITICAL] AES Key Source Unknown
   File: acr_decrypt_ls_ucode_ga100.c
   Issue: Key derivation not documented
   Impact: If hardcoded, break firmware confidentiality

4. [HIGH] Memory Training State Unverified
   File: fbflcn_gddr_boot_time_training_ga100.c
   Issue: Training results not cryptographically verified
   Impact: Fault injection => permanent memory corruption

5. [HIGH] Offset Validation Missing
   File: acr_wpr_ctrl_ga10x.c
   Issue: LSB offset written without bounds check
   Impact: Out-of-bounds firmware access, signature bypass

6. [MEDIUM] VREF Control Unchecked
   File: fbflcn_gddr_boot_time_training_ga100.c
   Issue: Voltage reference adjusted without range validation
   Impact: Hardware damage, side-channel leakage

7. [HIGH] Integer Overflow in Iteration
   File: acr_wpr_ctrl_ga10x.c
   Issue: i*2 in depMap loop can overflow
   Impact: Read from arbitrary memory location

8. [MEDIUM] Version Rollback Possible
   File: acr_wpr_ctrl_ga10x.c
   Issue: binVersion written without range check
   Impact: Load vulnerable old firmware
